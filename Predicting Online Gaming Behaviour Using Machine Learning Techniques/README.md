🧠 Proje Hedefleri
Bu veri seti ile aşağıdaki analizleri gerçekleştirebiliriz:

Oyuncu Etkileşimi Tahmini: Makine öğrenmesi modelleri kullanarak oyuncuların etkileşim seviyelerini tahmin etmek.
